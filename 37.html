Datenbank Nachhaltges Leipzig

Die Datenbank „Nachhaltiges Leipzig“ ist das aktuelle Gedächtnis für lokales Engagement zur nachhaltigen Entwicklung.

Mittels CMS dient die Datenbank der Pflegen von Projekten, Veranstaltungen, Aktionen und Bildungsangeboten zum Thema "Nachhaltige Entwicklung in Leipzig". Eine robuste API bietet Plattformen die Möglichkeit, diese Datenbestände zu verwerten.

Die Daten dieses Kooperationsprojekts der Leipziger Agenda 21 und der ZAK – Zukunftsakademie Leipzig e. V. werden auf den Seiten http://www.nachhaltiges-leipzig.de/ und dem Webauftritt der ZAK - ZukunftsAkademie - Leipzig e.V. abgebildet.

Features

  • API als Schnitstelle zu den Daten
  • Übersichtliches CMS
  • Ruby on Rails
  • Automatische Geolokalisierung
